# PocketMark Desktop

PocketMark Desktop is a local‑first Markdown note app for macOS.
This repository is used for release distribution only.

Download the latest `.dmg` from GitHub Releases and drag `PocketMark.app` into Applications.
Current builds are Apple Silicon (`aarch64`).
